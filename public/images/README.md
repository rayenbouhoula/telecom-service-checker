# Logo Placement

Place the Tunisie Telecom logo file here as:
- `tunisie-telecom-logo.png`

The logo should be:
- Format: PNG with transparent background
- Recommended size: 200x50px (or similar aspect ratio)
- Optimized for web (under 50KB if possible)

The logo will be displayed in:
1. Admin layout navbar/sidebar
2. Public layout navbar
3. Home page hero section
